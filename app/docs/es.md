Anuvaad es una aplicación de demostración que muestra cómo hacer que cualquier sitio web sea multilingüe utilizando Lingo.dev CLI y Lingo Engine.

## Características

- Traducción automática de textos de la interfaz de usuario
- Traducción automática de documentación
- Traducción en tiempo real para mensajes de chat
- Revisión de traducción con intervención humana

## Por qué es importante

La localización suele ser dolorosa y manual. Anuvaad automatiza todo el flujo de trabajo utilizando IA.
