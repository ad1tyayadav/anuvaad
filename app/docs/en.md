# Welcome to Anuvaad

Anuvaad is a demo application showing how to make any website multilingual using Lingo.dev CLI and Lingo Engine.

## Features

- Automatic translation of UI strings
- Automatic translation of documentation
- Real-time translation for chat messages
- Human-in-the-loop translation review

## Why it matters

Localization is usually painful and manual. Anuvaad automates the entire workflow using AI.
