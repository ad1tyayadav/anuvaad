# Bienvenue à Anuvaad

Anuvaad est une application de démonstration qui montre comment rendre n'importe quel site web multilingue en utilisant Lingo.dev CLI et Lingo Engine.

## Fonctionnalités

- Traduction automatique des chaînes d'interface utilisateur
- Traduction automatique de la documentation
- Traduction en temps réel pour les messages de chat
- Révision de traduction avec intervention humaine

## Pourquoi c'est important

La localisation est généralement pénible et manuelle. Anuvaad automatise l'ensemble du flux de travail en utilisant l'IA.
